#### How do unit tests help keep code clean?
They help keep code clean by encouraging the use of small functions with a single responsibility. Because they are easier to test, they are more readable, modular, and simpler to maintain.

#### What issues did you find while testing?
The tests identified unforeseen cases, such as dividing by zero, as well as poor or nonexistent error handling.
